Error Codes
-----------

A system error has occured whilst (error descriptor). Please seek help from the system adminstrator and quote the following error code: (error code).


1st character: -
    1 = File is located in ambassador Section
    5 = File is located in admin Section
    9 = File is located elsewhere


2nd character: -
    P = File type is PHP
    J = File type is JavaScript
    U = File type other than PHP or JavaScript


3rd - last but one character is the file acronymn e.g. UpdateEventVisibility would be UEV.


Last character indicates the error number in the file.
